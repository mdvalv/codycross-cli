# CodyCross CLI

## Usage

```
usage: main.py [-h] [-d] level puzzle question

CodyCross CLI

positional arguments:
  level                 level number
  puzzle                puzzle number
  question     question number, if 0, get answers for all questions

optional arguments:
  -h, --help            show this help message and exit
  -d, --debug           debug mode
```

## Example

```
$ python main.py 233 1 -q 1 -d
https://codycrossanswers.net/codycross-medieval-times-group-233-puzzle-1-answers
====================
Playground game aka Octopus tag Answers
> RED ROVER
====================
http://codycrossanswers.net/playground-game-aka-octopus-tag-answers
```
